.. _categories-density:

eContext\\Categories\\Density
=============================

Category Density search allows you to classify a keyword using a full-text Keyword index.  In cases where a particular
word is difficult to classify, you can obtain a list of eContext API Categories that are relevant to the keyword you're
curious about.
